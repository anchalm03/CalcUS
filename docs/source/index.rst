CalcUS - Quantum Chemistry Web Platform 
=============================================

**CalcUS** aims to democratize access to quantum chemistry by providing a user-friendly web interface to simplify running and analyzing quantum mechanical calculations. It is freely available, expandable and customizable. It promotes connectivity to multiple software packages and algorithms, thus providing state-of-the-art techniques to all practitioners.

For more information about the goal of the project, see the `paper <https://doi.org/10.1021/acs.jcim.1c01502>`_.

.. note::

   This project is under active development.
